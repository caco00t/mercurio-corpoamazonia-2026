CÓMO ABRIR EL VISOR
--------------------
OPCIÓN 1 (recomendada — funciona sin internet):
  1. Ejecutar "start_server.bat" con doble clic
  2. Abrir navegador (Chrome o Firefox)
  3. Ir a: http://localhost:8080
  4. Para cerrar el servidor: presionar Ctrl+C en la ventana negra

OPCIÓN 2 (doble clic directo):
  - Abrir "index.html" directamente (doble clic)
  - NOTA: En Chrome puede no cargar por restricciones CORS con file://
  - Funciona mejor en Firefox con configuración predeterminada

OPCIÓN 3 (VS Code):
  - Instalar extensión "Live Server"
  - Clic derecho en index.html > "Open with Live Server"

REQUISITOS
----------
  - Python 3.x instalado (para start_server.bat)
  - Navegador: Chrome 90+, Firefox 88+, Edge 90+
  - Conexión a internet solo para cargar el mapa base (OpenStreetMap/Satélite)
  - Los datos GeoJSON están EMBEBIDOS en el HTML — no requieren internet

FUNCIONALIDADES
---------------
  - Panel lateral: activar/desactivar capas por grupo
  - 3 mapas base: OpenStreetMap, Satélite (Esri), Topográfico
  - Clic en cualquier objeto: muestra sus atributos
  - Botón ⌂ (esquina superior derecha): zoom a extensión total
  - Leyenda interactiva (esquina inferior derecha)
  - Escala gráfica (esquina inferior izquierda)
  - Panel colapsable con botón ◀ ▶

-------------------
